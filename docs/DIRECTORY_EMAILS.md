# Emails ανά Διεύθυνση με Attachments

## Περίληψη

Δημιουργία και αποστολή αυτοματοποιημένων emails ανά **Διεύθυνση** όταν υπάρχουν νέες αιτήσεις στο ΣΗΔΕ. Κάθε email περιλαμβάνει:

1. **Προσωπικό περιεχόμενο** με τη μορφή:
   - Καλημέρα σας
   - Ενημέρωση για νέες αιτήσεις
   - Λεπτομέρειες κάθε αίτησης (διαδικασία, ημερομηνία, πρωτόκολλο, αιτών)
   - Αίτημα δράσης
   - Επικοινωνία

2. **Συνημμένα (Attachments)**:
   - `.zip` αρχείο με όνομα του πρώτου αιτούντος
   - Περιέχει τα αρχεία που ανέβασε ο αιτών (όταν υλοποιηθεί το download API)

3. **Προαιρετική αποστολή στο chat υποστήριξης** με σύνοψη

## Χρήση

### Δημιουργία & Αποστολή Emails ανά Διεύθυνση

```bash
python src/main.py --send-directory-emails
```

Δημιουργεί και αποστέλνει ένα email ανά Διεύθυνση με όλες τις νέες αιτήσεις που σχετίζονται με αυτήν.

### Με Αποστολή στο Chat Υποστήριξης

```bash
python src/main.py --send-directory-emails-to-chat
```

Επιπλέον αποστέλνει σύνοψη της δημιουργίας emails και της αποστολής στο group chat υποστήριξης.

## Διαμόρφωση

### Μεταβλητές Περιβάλλοντος (`.env`)

```env
# Email Configuration (υπάρχoντα)
EMAIL_ADDRESS=itsxed@rcm.gov.gr
EMAIL_PASSWORD=encrypted_password
EMAIL_SMTP_SERVER=mailsmtp.rcm.gov.ar
EMAIL_SMTP_PORT=587
EMAIL_USE_TLS=true

# Διευθύνσεις - Email διευθύνσεων (νέο)
DIRECTORY_EMAIL_ΔΙΕΥΘΥΝΣΗ_ΟΝΟΜΑΤΟΣ=email@directory.com
DIRECTORY_EMAIL_ΑΛΛΗ_ΔΙΕΥΘΥΝΣΗ=another@directory.com

# Teams/Chat Webhook (προαιρετικό)
TEAMS_WEBHOOK_URL=https://outlook.webhook.office.com/webhookb2/...
```

### Δυναμική Ανάθεση Email Διευθύνσεων

Η τρέχουσα υλοποίηση αναζητά περιβάλλοντες μεταβλητές όπως `DIRECTORY_EMAIL_{DIRECTORY_NAME}`.
Για παραγωγή, συνιστάται η αποθήκευση των email διευθύνσεων σε database ή config file.

## Αρχιτεκτονική

### Αρχείο: `src/directory_emails.py`

#### Κύριες Συναρτήσεις

| Συνάρτηση | Σκοπός |
|-----------|--------|
| `group_new_requests_by_directory()` | Ομαδοποιεί αιτήσεις ανά Διεύθυνση |
| `create_directory_eml()` | Δημιουργεί MIMEMultipart email object |
| `create_zip_with_attachments()` | Δημιουργεί zip με attachments |
| `attach_zip_to_email()` | Επισυνάπτει zip στο email |
| `send_directory_emails()` | Κύρια συνάρτηση αποστολής |

### Ροή Εκτέλεσης

```
1. Φόρτωση ημερήσιας αναφοράς (digest)
   ↓
2. Εξαγωγή νέων αιτήσεων (real_new + test_new)
   ↓
3. Ομαδοποίηση ανά Διεύθυνση
   ↓
4. Για κάθε Διεύθυνση:
   a. Δημιουργία HTML email
   b. Δημιουργία zip με attachments
   c. Αποστολή email
   d. (Προαιρετικά) Αποστολή στο chat
```
